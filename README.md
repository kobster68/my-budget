# My Budget

A simple mobile-first web app for tracking monthly spending.

**My Budget** helps users set category-based budgets, log expenses, and quickly see whether they are on track for the month.

## Tech Stack

**Frontend:** React, TypeScript, Tailwind CSS  
**Backend:** Python, FastAPI  
**Database:** PostgreSQL  
**DevOps/Deployment:** Docker, AWS Cloud Services, GitHub Actions

## Deployment

**My Budget** supports both local containerized development and remote AWS cloud deployment.

Local development uses Docker Compose with containerized frontend, backend, and PostgreSQL services.
